About This Project

🎯 Project Overview

The AI-Powered Drug Interaction Chatbot is an intelligent system that uses Natural Language Processing (NLP) and comprehensive drug databases to help users check potential interactions between medications.

💡 Problem Statement

Patients and healthcare providers often struggle to quickly check potential interactions between multiple medications, leading to adverse drug reactions and health risks. This chatbot provides instant, accurate information about drug interactions.

🔧 Technology Stack
  • Python 3.11
  • Flask Framework
  • NLP (spaCy, NLTK)
  • SQLite Database
  • Bootstrap 5
  • Machine Learning

⚠️ Disclaimer
This chatbot is for educational and informational purposes only. It should not replace professional medical advice. Always consult a healthcare provider or pharmacist before taking any medication.
📊 Data Sources

The drug interaction data used in this project is based on publicly available information and should not be considered clinically validated. In a production environment, this would be replaced with FDA databases or licensed medical data.
